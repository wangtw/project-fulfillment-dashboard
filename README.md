📘 系统使用说明书 (完全版 v20.3 Smart Sync Ed.)
🚀 项目履约监控看板操作指南
简单、直接、有效。拒绝废话。

1. 数据同步与更新 (NEW - Smart Sync)
智能合并逻辑： 导入 Excel 或 JSON 备份时，系统会自动识别 商机编号：
✅ 已存在： 自动更新进度、状态、备注（不会重复添加）。
➕ 不存在： 自动新增为新项目。
分发与汇总： 经理们可只填自己的数据，总监统一汇总时选择“智能合并”即可。

2. L-Level 资源优先级协议
P1 最高级 (Fire Fighting): 救火项目。单人持有 >2 个即视为高负载，建议分流。
P2 重点 (High): 核心交付项目，公司营收主力。
P3 正常 (Normal): 常规流程项目。
P4 挂起 (Suspended): 暂停。不占用当前资源，不计入 L-Load。
P5 终止 (Terminated): 项目死亡。

3. 每周填报 SOP (防健忘清单)
Step 1 查状态： 筛选你的名字，查看所有 [进行中] 项目。
Step 2 更延期： 只要本周进度滞后，立即增加 内因/外因延期数。不要等月底再算账。
Step 3 调级别： 如果项目变成烂摊子，把 L-Level 升为 P1；如果客户暂停，改为 P4。
Step 4 存备份： 点击顶部 ⚙️ 数据管理 -> [下载备份]。这步最重要！
